Blog Management System using Generative AI - Cosmo Blog

---

Overview

Cosmo Blog is a full-stack web application that allows users to create, manage, and explore blogs with a visually rich and interactive interface.
It is designed to provide a smooth blogging experience with modern UI, authentication, and content handling features.

---

Features

Blog Management

* Create, edit, and delete blog posts
* View all blogs in a structured layout
* Individual blog pages with detailed view

User Authentication

* User signup and login system
* Secure session handling
* Profile-based access

Modern UI/UX

* Glassmorphism-inspired design
* Animated and interactive components
* Responsive layout for all devices

Content Filtering System

* Detects inappropriate content
* Ensures clean and safe blog posts
* Custom filtering logic implemented

Dashboard & Analytics

* User dashboard interface
* Blog management overview
* Activity-based UI sections

Media & Assets

* Image support for blog posts
* Static assets handling
* Clean UI presentation
